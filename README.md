# TriviaClash 🎮

A modern gamified quiz/trivia Android application built with Kotlin, MVVM, Firebase, and Material Design 3.

## ✨ Features

- 🔐 **Authentication** - Email/Password, Google Sign-In, Anonymous, Facebook
- 🎯 **Quiz Gameplay** - Timed quizzes with multiple choice answers
- 🏆 **Gamification** - XP system, levels, coins, achievements
- 📊 **Leaderboard** - Global and weekly rankings
- 👤 **User Profile** - Statistics, match history, achievements
- 🔔 **Notifications** - Firebase Cloud Messaging
- 📈 **Analytics** - Firebase Analytics tracking
- 🌙 **Dark/Light Mode** - Theme switching
- 🌍 **Internationalization** - English and Macedonian language support
- 📱 **Responsive** - Phone, tablet, portrait and landscape support

## 🛠️ Tech Stack

- **Language:** Kotlin
- **Architecture:** MVVM
- **UI:** Material Design 3
- **Navigation:** Navigation Component
- **Local DB:** Room Database
- **Backend:** Firebase (Auth, Firestore, FCM, Analytics)
- **Async:** Kotlin Coroutines + StateFlow
- **Image Loading:** Glide

## 🔥 Firebase Services

- **Authentication** - Email, Google, Anonymous, Facebook
- **Firestore** - Users, Quizzes, Questions, Leaderboard, Achievements
- **Cloud Messaging** - Push notifications
- **Analytics** - User behavior tracking

## 🚀 Setup

1. Clone the repository
2. Open in Android Studio
3. Add your `google-services.json` file in the `app/` directory
4. Build and run
